## Module 14 Lesson 1: The many approaches to test planning

### Activity: Test Planning Exercise

#### Test Plan: LinkedIn Connection Groups

This is a fictional new feature for LinkedIn where users can create groups or lists of connections, for example grouping together recruiters, test experts, old colleagues and friends in their own list/group to make it easier to find people.

##### Stakeholders

- Developers
- Manager
- Product owner
- Tester
- Users (individuals, rather than businesses)

##### Scope

In Scope:

- CRUD of lists
- Handling of connection changes

Out of Scope:

- Connection management
- Timeline updates

##### Timeline

Q1 2026 working sprintly basis with incremental changes with a target release date of early Q2 2026.

Testing will be performed alongside development of new functionality

##### Test Types

- Automated E2E smoke tests
- Unit tests
- Hands on acceptance & explorary testing
- User journey testing when project is near completion

##### Risks

- Integration with other features may have unexpected consequences.
- Impact from other features changing.
